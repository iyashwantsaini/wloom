# wolwoloom

Editorial / typewriter Flutter design system. Mono typography, hairline borders, ink-on-paper palette with a periwinkle accent.

```dart
import 'package:wolwoloom/wolwoloom.dart';

MaterialApp(
  theme: WlmTheme.light(),
  darkTheme: WlmTheme.dark(),
  home: const MyPage(),
);
```

See the [project README](https://github.com/iyashwantsaini/WolwoLoom#readme) for the full component list and roadmap.

Run the example:

```sh
cd example
flutter run
```
